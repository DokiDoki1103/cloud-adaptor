### Rainbond Cloud Adaptor

该模块的作用是实现从 Kubernetes 集群到 Rainbond 集群的的创建和初始化。集群类型支持状态如下：

| Adaptor | 支持状态    |
| ------- | ----------- |
| ACK     | bate        |
| RKE     | bate        |
| TKE     | not support |
| Custom  | bate        |

### 版本对应关系

| ------- | -------- |
| V5.3    | V5.3     |
